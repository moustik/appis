1.1.1 usage

  1. /

    Affiche tout les membres de la base de données

  2. /member/<_id>

    Détails d'un membre à partir de son numéro d'adhérent

  3. /structure

    Affiche toutes les structures triées par identifiant de structure

  4. /structure/<structure>

    Liste l'ensemble des membres de la structure par son <identifiant de structure>

    • TODO [ ] afficher les structures enfants
  5. /request/<fmt>/<req>

    Execute une requete en base de donnée.

    • fmt : format de sortie html ou csv
    • req : de la forme /<collection>.<champ>=<type>:<valeur>
      • modèle de la base
    collections champs type exemple
    members _id i  
      nom s  
      prenom s  
      structure s  
      fonction s 210, 213, 220, 600,
    inscriptions ends s  
      inscriptiontype i 0[inscrit], 1, 2 [pre inscrit]
    formations name s TECH, APPRO, FIF,
      role s Formateur, Stagiaire
      date    
    diplomes name s BAFA, 2 Buchettes,
      date s  
    qualifications name s  
      titular s Oui
      obtained s JJ/MM/AAAA
      expires s JJ/MM/AAAA
    • type : s pour une chaine de caractère et i pour une valeur numerique
    • valeur : en toute lettre/chiffre avec les espaces

    Exemple : Tous les AP [code fonction 600] dont l'inscription est valide http://sgdf.nazkerin.org/request/html/members.fonction=s:600/inscriptions.inscription_type=i:0

1.2.2 mongoDB

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 7F0CEB10
echo "deb http://repo.mongodb.org/apt/debian wheezy/mongodb-org/3.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-3.0.list
sudo apt-get update
sudo apt-get install -y mongodb-org

# fix ovh kernel bug
sudo groupadd sysfs
sudo usermod -G sysfs mongodb
sudo chgrp sysfs /sys/devices/system/node
sudo chgrp sysfs /sys/dev/block/
sudo chgrp sysfs /sys/devices/virtual/block/md1/queue
sudo chmod g+rx /sys/devices/system/node
sudo chmod g+rx /sys/dev/block/
sudo chmod g+rx /sys/devices/virtual/block/md1/queue

sudo chgrp sysfs /sys/dev/block/8:1/
sudo chmod g+rx /sys/dev/block/8:1/
sudo chgrp sysfs /sys/kernel/mm/
sudo chmod g+rx /sys/kernel/mm/
sudo chgrp sysfs
sudo chmod g+rx
